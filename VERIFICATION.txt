Testing Checklist:
==================
To verify everything works:

1. Start server:
   npm start

2. Test URL audit:
   - Enter: https://www.w3.org
   - Should complete and show results

3. Test HTML audit with issues:
   - Paste incomplete HTML with accessibility problems
   - Should show violations with fix suggestions

4. Test perfect HTML:
   - Paste accessible HTML
   - Should show score 100 with "No violations found"

5. Test error handling:
   - Invalid URL should show error
   - Large content should show error
   - Missing input should show error

6. Test report download:
   - Click "Download Report as HTML"
   - Should save file with timestamp

7. Test responsive design:
   - Resize browser
   - Mobile view should work properly

8. Test keyboard support:
   - Enter key should submit form
   - Ctrl+Enter in HTML input should submit
